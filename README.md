# טיימר בשרי-חלבי - אפליקציית PWA 🍗🥩

## מה זה?
זו אפליקציה מלאה (PWA - Progressive Web App) שעובדת כמו אפליקציה רגילה:
- ✅ אייקון יפה במסך הבית
- ✅ פותחת בחלון נפרד (ללא סרגל דפדפן)
- ✅ עובדת גם במצב אופליין
- ✅ מהירה ומקצועית

## איך להתקין?

### אפשרות 1: העלאה לאינטרנט (מומלץ!)

#### שיטה A: Netlify (הכי פשוט - 2 דקות!)
1. כנס ל: https://app.netlify.com/drop
2. גרור את **כל התיקייה** (pwa-app) לחלון הדפדפן
3. קבל קישור (כמו: https://your-app.netlify.app)
4. פתח את הקישור בנייד
5. תראה הודעה "התקן את האפליקציה" - לחץ התקן!
6. האפליקציה תופיע במסך הבית עם האייקון היפה! 🎉

#### שיטה B: GitHub Pages
1. צור repository חדש ב-GitHub
2. העלה את כל הקבצים
3. Settings → Pages → Source: main branch
4. קבל קישור והתקן כמו בשיטה A

### אפשרות 2: שרת מקומי לבדיקה
1. שים את כל הקבצים באותה תיקייה
2. פתח Terminal בתיקייה
3. הרץ: `python3 -m http.server 8000`
4. פתח בדפדפן: http://localhost:8000
5. תראה הודעה "התקן את האפליקציה"

## מה יש בתיקייה?
- `index.html` - הדף הראשי
- `app.js` - קוד האפליקציה
- `manifest.json` - הגדרות PWA (שם, אייקון, צבעים)
- `service-worker.js` - מאפשר עבודה אופליין
- `icon-192.png` - אייקון קטן
- `icon-512.png` - אייקון גדול

## תכונות האפליקציה
✨ **טיימרים:**
- טיימר עוף (1-6 שעות, ברירת מחדל: 5)
- טיימר בקר (1-6 שעות, ברירת מחדל: 6)

⚙️ **הגדרות:**
- בחירת סוג התראה (ללא / רטט / צלצול / שניהם)
- התאמת זמני המתנה בקפיצות של חצי שעה

📱 **חכם:**
- שומר את הטיימר גם אם סוגרים את האפליקציה
- עובד גם ללא אינטרנט (אחרי ההתקנה הראשונה)
- מוצג השעה בה תהיה חלבי

## שאלות נפוצות

**ש: למה הוודעה "התקן את האפליקציה" לא מופיעה?**  
ת: PWA עובד רק דרך HTTPS או localhost. קבצים מקומיים (file://) לא נתמכים.

**ש: איך אני מסיר את האפליקציה?**  
ת: כמו כל אפליקציה - לחיצה ארוכה על האייקון ← הסר

**ש: זה עולה כסף?**  
ת: לא! Netlify ו-GitHub Pages חינמיים לחלוטין.
