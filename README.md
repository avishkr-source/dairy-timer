## ✨ חדש בגרסה זו!

1. ✅ **העיגולים אחד ליד השני** - כמו שביקשת, ללא גלישה למסך חדש
2. ✅ **הטיימר הורם למעלה** - לא מסתיר את פסוק התורה למטה

5. ✅ **רקע חדש** - התמונה המקסימה שלך של האדם עם הסטייק והקפה!

- ☑️ הגדרות פשוטות - צלצול ורטט בנפרד
- ⏰ טיימר גדול וברור (48px)
- 📖 פסוק תורה בגופן מיוחד למטה

## 🚀 איך להתקין?

### העלאה ל-GitHub (עדכון):

**אם כבר יש לך repository:**

#### דרך הממשק (הכי פשוט):
1. כנס ל-repository שלך ב-GitHub
2. לחץ על `index.html`
3. לחץ על העיפרון ✏️ (Edit)
4. מחק הכל, העתק והדבק את התוכן החדש מ-`index.html`
5. Commit changes
6. חזור ל-repository
7. לחץ על `app.js` וחזור על התהליך
8. סיימת! 🎉

#### דרך Git Terminal:
```bash
cd path/to/your/repo
# העתק את הקבצים החדשים לתיקייה
git add .
git commit -m "עדכון גרסה 3 - רקע חדש והודעות משופרות"
git push
```

### התקנה חדשה ב-Netlify:
1. כנס ל: https://app.netlify.com/drop
2. גרור את התיקייה `pwa-app-v3`
3. קבל קישור והתקן בנייד!

## 🎯 פתרון בעיות

**הרקע לא מופיע?**
- התמונה מוטמעת ב-HTML, אמורה לעבוד תמיד
- אם לא, נקה cache של הדפדפן

**העדכון לא נראה ב-GitHub Pages?**
- המתן 1-2 דקות
- נקה cache (Ctrl+Shift+R)

**האפליקציה לא מתעדכנת בנייד?**
- מחק את האפליקציה הישנה
- התקן מחדש מהקישור

## קבצים בתיקייה
- `index.html` - דף ראשי עם רקע מוטמע
- `app.js` - לוגיקה עם הודעות חכמות
- `manifest.json` - הגדרות PWA
- `service-worker.js` - תמיכה במצב אופליין
- `icon-192.png` + `icon-512.png` - אייקונים
