# טיימר בשרי-חלבי - אפליקציית PWA מעוצבת 🍗🥩

## מה חדש בגרסה זו? ✨

### עיצוב מחודש:
- 🎨 **רקע מותאם אישית** - תמונת הרקע שלך עם אפקט blur מקצועי
- 🌟 **כפתורים בצבע חום בהיר** - נראים חמים ומזמינים
- ✅ **כפתור פעיל בירוק** - ברור מאוד מה פועל
- 💡 **שינוי רקע בזמן הטיימר** - הרקע מתבהר כשהטיימר רץ
- 📏 **אייקונים גדולים יותר** - קל יותר לראות
- 🔤 **טיימר מוגדל** - ספרות גדולות וברורות (48px)
- 📖 **פסוק תורה בתחתית** - "לֹא תְבַשֵּׁל גְּדִי בַּחֲלֵב אִמּוֹ" בגופן כתב

### כותרת חכמה:
- ❓ "מה אכלת?" - מופיעה בהתחלה
- 🫥 נעלמת כשמפעילים טיימר
- 🔙 חוזרת כשהטיימר מסתיים

### הגדרות מפושטות:
- ☑️ **תיבות סימון פשוטות** - צלצול ורטט (ללא ברירת מחדל)
- 🎯 כל אחד עובד בנפרד - אפשר לבחור אחד, שניהם, או אף אחד
- ⚙️ עיצוב נקי ואינטואיטיבי

## איך להתקין?

### הדרך הכי פשוטה - Netlify (2 דקות):

1. **כנס ל:** https://app.netlify.com/drop
2. **גרור את התיקייה** `pwa-app-v2` לחלון הדפדפן
3. **קבל קישור** (משהו כמו: `https://your-timer.netlify.app`)
4. **פתח בנייד** את הקישור
5. **לחץ "התקן"** או "הוסף למסך הבית"
6. **נהנה מהאפליקציה!** 🎊

### אפשרויות נוספות:
- **GitHub Pages** - להעלאה לרפוזיטורי
- **Vercel** - פריסה מהירה
- **שרת מקומי** - `python3 -m http.server 8000`

## תכונות האפליקציה

✨ **טיימרים מותאמים:**
- 🍗 טיימר עוף (1-6 שעות)
- 🥩 טיימר בקר (1-6 שעות)
- ⏱️ קפיצות של חצי שעה

⚙️ **התראות גמישות:**
- 🔔 צלצול (אופציונלי)
- 📳 רטט (אופציונלי)
- 🎯 כל אחד בנפרד

🎨 **עיצוב מקצועי:**
- 🖼️ רקע מותאם אישית
- 🌈 אנימציות חלקות
- 📱 ממשק נקי ופשוט
- 📖 פסוק תורה בגופן מיוחד

💾 **חכם ונוח:**
- שומר טיימר פעיל
- עובד אופליין
- מציג שעת סיום
- ממשק בעברית

## קבצים בתיקייה
- `index.html` - הדף הראשי (עם רקע מוטמע)
- `app.js` - לוגיקת האפליקציה
- `manifest.json` - הגדרות PWA
- `service-worker.js` - תמיכה במצב אופליין
- `icon-192.png` - אייקון קטן
- `icon-512.png` - אייקון גדול

## טיפים

- אפשר התראות בדפדפן
- השתמש במצב מסך מלא
